## A cli clipboard tool 

#### Install 

`npm install klipboard -g`

#### Usage

``` 
//cache your item 
klip jquery https://cdnjs.cloudflare.com/ajax/libs/jquery/3.0.0-alpha1/jquery.min.js
//get your item to clipboard,then you can 'CMD + V'
klip jquery 
```
